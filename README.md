## Descripción
Este es el repositorio de los sprints realizados para el módulo DWES de 2º año de DAW.

---

## Estado del proyecto
![Estado](https://img.shields.io/badge/Estado-WIP-yellow)

Terminado el sprint 4 de Django.  
Estructura de carpetas:
- defensaSprints: archivos de la defensa de los sprints de PHP.
- fpdf186: archivos de la práctica opcional de fpdf.
- opcional_pdf: archivos resultado y recursos utilizados con el creador de PDF.
- Carpetas de los sprints:
  - sprint0python
  - sprint1apache
  - sprint2phpsql
  - sprint3django
  - sprint3sesiones

## Funciones y aplicaciones
### Sprint 0
Diferentes ejercicios de Python.
### Sprint 1
Página web de ejemplo y varios ejercicios de PHP desplegados con Apache.
### Sprint 2
Archivos de página web con PHP para acceder a una base de datos.
### Sprint 3 Sesiones
Continuación de la página web añadiendo usuarios a la base de datos y a la funcionalidad de la web.
### Sprint 3 Django
Creación de una RestAPI a partir de la base de datos de la web anterior.
